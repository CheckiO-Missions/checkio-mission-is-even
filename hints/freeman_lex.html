Here it is. 
def is_even(num: int) -> bool:
    
    return num % 2 == 0
The solution may be a little bit tricky. (num % 2) already returns value, which may be used as bool, so we don't need (==0). But for even numbers (num % 2) will return 0, which in boolean logic means False. So you have to use 'not' operator to invert bool value, because the function must return True for even. And the code is: 
def is_even(num: int) -> bool:
                
    return not num % 2
