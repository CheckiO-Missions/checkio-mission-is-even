Here it is. 
function isEven(num: number): boolean {

    return num % 2 == 0;
}

The solution may be a little bit tricky. num % 2 already returns value, which may be used as bool, so we don't need == 0. But for even numbers num % 2 will return 0, which in boolean logic means false. So you have to use 'not' operator ! to invert bool value, because the function must return true for even numbers. And the code is: 

function isEven(num: number): boolean {

    return !(num % 2);
}
