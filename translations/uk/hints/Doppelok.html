З чого мені розпочати вирішувати цю місію?
Ідея місії в тому, щоб перевірити чи є дане число парним за допомогою умови "if". Отже, якщо умова "if" виконується, то тобі потрібно повернути "True", якщо ні - "False"
Як мені перевірити, що число парне?
Ти можеш використати оператор "%", щоб перевірити число на парність. Спробуй у консолі: 
number = 10
number % 2  # оператор "%" допоможе зрозуміти чи ділиться число "number" на 2 без остачі
0  # якщо так, то ти отримаєш "0", якщо ні - "1"
Потрібна допомога! Що далі роботи?
Спробуй поєднати умову "if" і оператор "%". Але тут треба приділити увагу! Оператор "%" поверне "0" (це те саме що "False") якщо "number" парне число.
Як мені це зробити? Покажи код, будь ласка!
Тримай: 
if not number % 2  # це те саме що і "if number % 2 == 0"
    return True  # ти повертаєш "True" якщо число парне
else:
    return False  # ти повертаєш "False" якщо число непарне
Останнє запитання. Як виглядає фінальний код? 
def is_even(num: int) -> bool:
    if not num % 2:
        return True
    else:
        return False

Або можна записати в один рядок:

def is_even(num: int) -> bool:
    return True if not num % 2 else False
