Я уявлення не маю, як розв'язати це завдання!
Парні числа діляться на 2 із залишком 0. Тому, щоб розв'язати цю місію, ти маєш поділити вхідне число на 2, знайти залишок і порівняти його із 0.
Які інструменти Python мені для цього використати?
Тобі слід використати оператор остачі (%), який повертає остачу від ділення і оператор рівності (==).
Я вцілому розумію, але потрібно побачити остаточний код.
Ось він: 
def is_even(num: int) -> bool:
    
    return num % 2 == 0
Розв'язок може бути цікавішим. (num % 2) відразу повертає значення, яке може розглядатися як булеве, тому нам не потрібне (==0). Але для парних чисел (num % 2) повертатиме 0, що в булевій логіці значить False. Тому тобі потрібно використати оператор 'not' для заміни булевого значення на протилежне, оскільки функція повинна повертати True для парних чисел. У такому випадку код матиме вигляд: 
def is_even(num: int) -> bool:
                
    return not num % 2
