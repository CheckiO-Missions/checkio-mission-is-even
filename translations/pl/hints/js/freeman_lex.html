Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

liczby parzyste są podzielne przez 2 (dzielą się przez 2 z resztą 0). Aby rozwiązać to zadanie, należy podzielić liczbę wejściową przez dwa, znaleźć resztę i porównać ją z 0.

Od czego powinienem zacząć?

Powinien Pan użyć funkcji modulus operator %, która zwraca resztę z dzielenia, oraz operatora równości (==).

Ogólnie rozumiem, ale proszę zobaczyć ostateczne rozwiązanie.

Proszę. 
function isEven(num: number): boolean {

    return num % 2 == 0;
}

Rozwiązanie może być nieco skomplikowane. num % 2 już zwraca wartość, która może być użyta jako bool, więc nie potrzebujemy == 0. Ale dla liczb parzystych num % 2 zwróci 0, co w logice boole'owskiej oznacza false. Proszę więc użyć operatora 'not' !, aby odwrócić wartość bool, ponieważ funkcja musi zwrócić true dla li

czb parzystych.

Kod wygląda następująco: 

function isEven(num: number): boolean {

    return!(num % 2);
}

Dzięki. Teraz wiem i rozumiem więcej! Wszystko działa!
