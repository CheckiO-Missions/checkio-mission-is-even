Nie mam pojęcia, jak zacząć rozwiązywać tę misję

liczby parzyste są podzielne przez 2 (dzielą się przez 2 z resztą 0). Aby rozwiązać to zadanie, należy podzielić liczbę wejściową przez dwa, znaleźć resztę i porównać ją z 0.

Jakich narzędzi Pythona powinienem użyć?

Powinien Pan użyć funkcji modulo operator %, która zwraca resztę z dzielenia, oraz operatora równości (==).

Ogólnie rozumiem, ale proszę zobaczyć ostateczne rozwiązanie.

Proszę. 
def is_even(num: int) -> bool:
    
    return num % 2 == 0

Rozwiązanie może być nieco skomplikowane. num % 2 już zwraca wartość, która może być użyta jako bool, więc nie potrzebujemy == 0. Ale dla liczb parzystych num % 2 zwróci 0, co w logice boolean oznacza False. Proszę więc użyć operatora not do odwrócenia wartości bool, ponieważ funkcja musi zwrócić True dla par

zystości.

Kod wygląda następująco: 

def is_even(num: int) -> bool:
                
    return not num % 2

Dzięki. Teraz wiem i rozumiem więcej! Wszystko działa!
