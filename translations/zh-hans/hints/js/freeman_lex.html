这就是了。 
function isEven(num: number): boolean {

    return num % 2 == 0;
}

这个解决方案可能有点棘手。num % 2 已经返回值，可以作为bool使用，所以我们不需要== 0 。但是对于偶数，num % 2 将返回0 ，这在布尔逻辑中意味着false 。所以你必须使用'not'操作符! 来反转bool值，因为对于偶数，该函数必须返回true 。而代码是： 

function isEven(num: number): boolean {

    return!(num % 2);
}
