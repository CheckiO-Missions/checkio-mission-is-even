💡 Gerade Zahlen sind gleichmäßig durch 2 teilbar (geteilt durch 2 mit Rest 0). Um diese Aufgabe zu lösen, muss man also die eingegebene Zahl durch zwei teilen, den Rest finden und mit 0 vergleichen.

Welche Python-Instrumente sollte ich verwenden?

Sie sollten den Modulo-Operator % verwenden, der den Rest der Division liefert, und den Gleichheitsoperator (==).

Ich verstehe das im Allgemeinen, muss aber die endgültige Lösung sehen.

Hier ist sie. 
def is_even(num: int) -> bool:
    
    return num % 2 == 0

Die Lösung könnte etwas knifflig sein. num % 2 gibt bereits einen Wert zurück, der als bool verwendet werden kann, also brauchen wir nicht == 0. Aber für gerade Zahlen gibt num % 2 0 zurück, was in der booleschen Logik Falsch bedeutet. Sie müssen also den Operator not verwenden, um den boolschen Wert zu invertieren, da die Funktion für gerade Zahlen True zurückgeben muss. Und der Code ist: 

def is_even(num: int) -> bool:
                
    return not num % 2
