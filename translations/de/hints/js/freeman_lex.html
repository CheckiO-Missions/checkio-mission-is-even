💡 Gerade Zahlen sind gleichmäßig durch 2 teilbar (geteilt durch 2 mit Rest 0). Um diese Aufgabe zu lösen, musst du also die eingegebene Zahl durch zwei teilen, den Rest finden und mit 0 vergleichen.

Womit soll ich beginnen?

Sie sollten den Modulus-Operator % verwenden, der den Rest der Division liefert, und den Gleichheitsoperator (==).

Ich verstehe das im Allgemeinen, muss aber die endgültige Lösung sehen.

Hier ist sie. 
function isEven(num: number): boolean {

    return num % 2 == 0;
}

Die Lösung kann etwas knifflig sein. num % 2 gibt bereits einen Wert zurück, der als bool verwendet werden kann, also brauchen wir nicht == 0. Aber für gerade Zahlen gibt num % 2 0 zurück, was in der boolschen Logik falsch bedeutet. Sie müssen also den 'not'-Operator ! verwenden, um den boolschen Wert zu invertieren, da die Funktion für gerade Zahlen true zurückgeben muss. Und der Code ist: 

function isEven(num: number): boolean {

    return !(num % 2);
}
